## 4. Elements

All backends MUST support the following core elements.

### 4.1 Text

Represents read-only text.

Required properties:
- text: string

### 4.2 Image

Represents a raster or vector image.

Required properties:
- source: URI or backend-defined handle

### 4.3 TextField

Represents editable text input.

Required properties:
- value: string
